SynchWeb

Experimental Information Management for Synchrotrons and Large Scale Facilities

Download
Virtual Machine Setup

SynchWeb1 is the most advanced interface to the ISPyB2 database, it captures virtually all aspects of an experiment and presents them via a friendly web interface. Designed for real-time monitoring of synchrotron experiments and remote access, project evaluation, and long term facility reporting

But I Don't Work In MX?

Full-featured

  • Real-time Monitoring

    SynchWeb updates in real time showing you what is going on during your experiment

  • Staff Tools

    Extensive tools for beamline staff to keep track of whats going on and wrong, and evaluate beamline usage

  • Fully Integrated

    Results and processing outputs are fully integrated, along with tools for evaluating results

  • Optimised

    Optimised for performance on all types of devices. Source code is lazy loaded and views are paginated

  • Barcode Scanning App

    Complementary iOS & Android barcode scanner application for goods handling and shipment tracking

  • Mobile Ready

    SynchWeb is mobile ready out of the box supporting some 5% of our users

For Discplines Outside Macromolecular Crystallography

SynchWeb is rapidly expanding to cover non MX disciplines. Small Molecule Crystallography, Powder Diffraction, Tomography, and Electron Microscopy all have basic support and can be used for sample shipping and data collection monitoring.
